## Project Overview 
* The platform is a database-driven web application for local clothing vendors to sell products online. 
* Customers can browse products, place orders, and perform secure electronic payments. 

### Objectives 
* Secure user registration, login, and profile management. 
* Admin capability to manage (add, edit, delete, categorize) products with images. 
* User-friendly shopping cart and secure checkout interface. 
* Integration of electronic payment methods (M-Pesa, EcoCash). 

### Project Triple Constraints 
* **Scope:** Web-based platform with product catalog, user registration, cart, and payment integration. 
* **Time:** Duration of 3 months. 
* **Cost:** Estimated budget of M 2000.00. 

### Key Assumptions
* System access via web browsers. 
* Third-party payment services (M-Pesa, EcoCash) will remain operational. 
* Vendors provide accurate product data/images and users have internet access.
